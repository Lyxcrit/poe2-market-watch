# Safety and Terms

This tool must remain manual-only decision support for market snapshot analysis.

It must not add:

- Game-client automation.
- Memory reading.
- Screen scraping.
- OCR-based trade discovery.
- Click automation.
- Trade automation.
- Undocumented endpoint scraping.
- Client hooks or game-file inspection.

The tool only uses public or third-party snapshot market data, local static
gold-cost data, and terminal output.

Every trade leg must be manually validated in-game by the user before execution.
Snapshot data is not live order-book depth and may be stale by the time it is
displayed.
